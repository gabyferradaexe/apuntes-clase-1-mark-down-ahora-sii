1) Telar de Jacquard (1801)  creada por **Joshep Marie Jacquard**

- Fue el primer sistema de almacenamiento de info **_binaria_**
- El telar usaba un sistema de **tarjetas perforadas** para lograr diseños o dibujos completos y automatizados 




1) Máquina diferencial (1822)**1**    creada por **Charles babbage**

Diseñó esta máquina para sistematizar el cálculo y la tabulación de funciones polinómicas, tareas que hasta entonces se realizaban manualmente y eran propensas al error. Era una maquina de gran escala, compuesta por engranajes que servian para resolver ecuaciones e imprimir los resultados en fomra de tablas de matematicas.
