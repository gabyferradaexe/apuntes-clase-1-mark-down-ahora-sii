1) ##Telar de Jacquard (1801)  creada por **Joshep Marie Jacquard**

- Fue el primer sistema de almacenamiento de info **_binaria_**
- El telar usaba un sistema de **tarjetas perforadas** para lograr diseños o dibujos completos y automatizados.

a) **SI Habia agujero** : Primero la varilla pasaba, el gancho se activaba y levantaba el hilo (se le puede llamar 1 o "Encendido")

b) **Si NO habia agujero** : La varilla chocaba contra la tarjeta, el gancho no se movia , y el hilo se quedaba por debajo (se le puede llamar 0 o "apagado")

Este movimiento busca garantizar ##4 LIBERTADES ESENCIALES

a) Libertad 0 (**Uso**) ----> Ejecutar el programa como se desee con cualquier proposito

d) Libertad 3 (**Mejora**) ----> Libertad de mejorar el programa y hacer publicas las mejoras , para que la comunidad se beneficie. Acceso a codigo fuente es necesario.
