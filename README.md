1) Telar de Jacquard (1801)  creada por **Joshep Marie Jacquard**

- Fue el primer sistema de almacenamiento de info **_binaria_**
- El telar usaba un sistema de **tarjetas perforadas** para lograr diseños o dibujos completos y automatizados.

a) **SI Habia agujero** : Primero la varilla pasaba, el gancho se activaba y levantaba el hilo (se le puede llamar 1 o "Encendido")

b) **Si NO habia agujero** : La varilla chocaba contra la tarjeta, el gancho no se movia , y el hilo se quedaba por debajo (se le puede llamar 0 o "apagado")

2) Máquina diferencial (1822) **1**    creada por **Charles babbage**

Diseñó esta máquina para sistematizar el cálculo y la tabulación de funciones polinómicas, tareas que hasta entonces se realizaban manualmente y eran propensas al error. Era una maquina de gran escala, compuesta por engranajes que servian para resolver ecuaciones e imprimir los resultados en forma de tablas de matematicas.

  ___  

3) Maquina Analitica (1834-1837)  creada por **Charles babbage** tambien

Se considera la **PRIMERA MAQUINA De Calculo AUTOMATICA** , fue diseñada para realizar cualquier tipo de calculo ya establecido anteriormente Y para tener una capacidad aun mayor que la maquina diferencial 1 original.

- Donde se hacen las operaciones ----> En el **procesador** o **CPU** ( el molino o the mill)
- Donde se guardan los numeros y datos ----> En la **memoria** (el almacen o the store)
- Donde es la entrada de datos e instrucciones ----> En el **software** (tarjetas perforadas o punch card system)
